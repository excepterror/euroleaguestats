## EuroLeagueStats (ELS)
<img align="right" width="256" height="256" alt="Logo" src="https://github.com/user-attachments/assets/4452e256-d2f6-4dcf-b7f5-09100f450694" />
EuroLeagueStats
is an application for Android devices.

The code is written in Python 3.12 and the UI is designed with [kivy](https://kivy.org/#home). 

<p>The purpose of the application is to provide a fast & elegant alternative
to those who wish to have access to statistical data with regard to the performance
of all professional basketball players who participate in Euroleague, the annual top European basketball competition.</p>

### Features
For a detailed description and more info, please visit [the app on Google store](https://play.google.com/store/apps/details?id=org.euroleaguestats.euroleaguestats).
